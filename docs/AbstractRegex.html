Detailed Description

A base class for regex that defines match, just in case some other algorithm (say prefix matching) needs to be used instead of the full regular expression library. I originally used it for some performance tuning, but it ended up not making much of a difference, especially if you use the ^ and $ regular expression operators to mark the beginning and the end of a string.
